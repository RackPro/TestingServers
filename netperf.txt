apt install netperf
netperf --help
-------------------------------
1) Запуск севрера netperf на одной машине:
netperf -s
2) Запуск клиента netperf на другой машине:
netperf -H <IP_адрес_сервера> -t <тип_теста>
-------------------------------

-H <IP_адрес_сервера>: Указывает IP-адрес сервера.
-t <тип_теста>: Выбирает тип теста:
TCP_STREAM: Измеряет пропускную способность непрерывного потока данных TCP.
TCP_RR: Измеряет пропускную способность TCP с использованием запросов-ответов (Round-Robin).
TCP_MAERTS: Измеряет пропускную способность TCP с использованием многопоточных запросов-ответов.
TCP_CRR: Измеряет пропускную способность TCP с использованием конкурентных запросов-ответов.
UDP_STREAM: Измеряет пропускную способность непрерывного потока данных UDP.
UDP_RR: Измеряет пропускную способность UDP с использованием запросов-ответов (Round-Robin).
-l <продолжительность>: Задает продолжительность теста в секундах.
-r <количество_повторов>: Указывает количество повторов теста.
-p <порт>: Указывает порт для соединения (по умолчанию - 2000).
-m <размер_пакета>: Устанавливает размер пакета в байтах.
-s <количество_потоков>: Задает количество потоков для теста.
-b <скорость_передачи>: Устанавливает скорость передачи данных в Мбит/с.

Анализ результатов:
Throughput: Показывает максимальную скорость передачи данных между клиентом и сервером.
Minimum, Maximum, Average: Измеряют задержку сети, определяя минимальное, максимальное и среднее время отклика.
Lost: Позволяет оценить количество потерянных пакетов во время теста.
RetransSegs: Показывает количество переотправленных сегментов, что может указывать на проблемы с сетью.
